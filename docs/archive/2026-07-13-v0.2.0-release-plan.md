## 停止条件

- full gateがgreenでない。
- tarballに秘密、browser profile、state、fixture、RAG、research scriptが混入する。
- npm identityまたはregistryが想定外。
- GitHub remote／branchが想定外。
- npm上に`0.2.0`が既に存在し、local内容と同一と確認できない。

## rollback

- GitHub push後の問題は追加revert commitで戻す。履歴改変やforce pushはしない。
- npm versionは上書き不可。問題時はdeprecateし、修正版を次patchとして公開する。
- global installは`npm uninstall -g gpt-connector`または既知version再installで戻す。

## 実行中の失敗記録

- 初回`npm publish`は公開前の`prepublishOnly`で停止した。隔離tarballを展開した`.gpt-connector-tmp/**`をESLintが走査し、生成済みJS／d.tsへ42 errorsを報告した。
- npm registry read-backは`0.1.0`のみで、`0.2.0`未公開を確認した。
- 根本修正としてgitignore済みrelease作業directoryをESLint ignoreへ追加し、展開物を残したままfull gateを再実行してgreenを確認した。
